Frequently Asked Questions

Can I import my Notion workspace?

Yes! Export your Notion workspace as Markdown and import directly into {{ brand_name }}. Most content transfers cleanly, though some Notion-specific features like databases won't have exact equivalents.

How is {{ brand_name }} faster than Notion?

{{ brand_name }} uses viewport-based rendering and CRDTs for real-time sync. We only load what's visible on screen, so document size doesn't affect performance. Notion loads more data upfront, which causes slowdowns with large workspaces.

Does {{ brand_name }} have databases like Notion?

{{ brand_name }} focuses on text-first knowledge management. If you heavily rely on Notion's database features with relations and rollups, Notion might be a better fit. We prioritize speed and simplicity over complex data structures.

Is {{ brand_name }} really open source?

Yes! We use the Elastic License. You can view the source code, self-host for your team, and audit how everything works. The license protects against competitors reselling the product while keeping it free for users.
